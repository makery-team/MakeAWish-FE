## 🚀 빠른 시작 (Quick Start)

1. **환경 변수 설정**: `.env.example` 파일을 복사하여 `.env` 파일을 생성합니다.
2. **패키지 설치**: `npm install`
3. **프로젝트 실행**: `npx expo start`
